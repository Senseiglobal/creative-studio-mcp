# Creative Studio MCP Feedback Guide

Thank you for helping improve Creative Studio MCP.

This project is built for creative professionals, freelancers, studios, agencies, print shops, virtual assistants, marketing teams, and small businesses.

## How To Share Feedback

Use GitHub Discussions for general conversation:

https://github.com/Senseiglobal/creative-studio-mcp/discussions

Use GitHub Issues for clear bugs, setup problems, or specific feature requests:

https://github.com/Senseiglobal/creative-studio-mcp/issues

Sponsor the project:

https://github.com/sponsors/Senseiglobal

## Feedback We Are Looking For

- What creative work do you do?
- What business tasks take too much time?
- What kind of quotes do you create most often?
- Which services should be added to the default pricing list?
- What pricing ranges should be improved?
- What should the client database do first?
- What part of setup is confusing?
- What examples would help beginners more?

## Community Guidelines

- Be clear and respectful.
- Share practical examples when possible.
- Do not post private client information.
- Do not share API keys, passwords, wallet seed phrases, or private business data.
- Keep feedback focused on improving the project.
