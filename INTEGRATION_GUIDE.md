## Website Integration (Simple Version)

### What This Does

A visitor comes to your website and sees:

```
Chat with our AI assistant →
"What services do you offer?"
[instant list of your services]

"Can you give me a quote for branding?"
[AI asks a few quick questions]
[professional quote appears instantly]

"Send me this quote"
[quote is emailed to them]
```

All automatic. You don't have to do anything.

---

### How to Set It Up

If you have a website built with:
- **Wix, Squarespace, Shopify, WordPress** → Use their built-in chatbot feature
- **Custom website** → See your web developer (share this document)
- **No website yet** → Use Google Sites (free) and add a chatbot

---

### Step-by-Step (Non-Technical Version)

1. **Get a chatbot service** (free options):
   - ChatGPT (openai.com): $20/month for best features
   - Claude (claude.ai): Free
   - Microsoft Copilot: Free

2. **Tell the chatbot about your tool:**
   Write this in the chatbot's instructions:
   ```
   "You have access to a tool that generates creative services quotes, 
   calculates payments, and creates project checklists. Use these tools 
   to help customers."
   ```

3. **Add the chatbot to your website** (each platform is different):
   - **Wix:** Go to Add → Chat → Chat by Wix
   - **Squarespace:** Add a Chat Block
   - **WordPress:** Use a chat plugin like Tidio
   - **Custom site:** Ask your web developer

---

### You Can Also Create a Quote Form

## WhatsApp Business Integration

### How It Works

A client messages you on WhatsApp:

```
Client: "Hi! How much for branding?"

You (using this tool): 
"Brand Identity starts at $500 with 70% upfront.
Want a custom quote?"

Client: "Yes, send me one"

You (using Claude/ChatGPT):
"Create a quote for Sarah for Brand Identity at $3,000"
[Tool generates quote]
You: [paste the quote into WhatsApp]
```

### To Set This Up

1. Get WhatsApp Business (free app)
2. Install this tool and Claude/ChatGPT
3. When a client asks for a quote, use Claude/ChatGPT to generate one
4. Copy and paste it into WhatsApp

**No complex setup needed.** It's manual, but fast.

## Email Integration (Advanced)

### How It Works

1. Client fills out form on your website
2. Their info goes to your email
3. You copy-paste their info into Claude/ChatGPT
4. Tool generates a professional quote
5. You email the quote to them

**Time saved:** 15 minutes per quote → 1 minute per quote

### To Set This Up Fully (With Your Web Developer)

```
Website form submission
↓
Sends data to your backend
↓
Calls the creative-studio-mcp tool
↓
Generates quote automatically
↓
Emails quote to client
↓
Sends confirmation to you
```

This requires someone who can code. Ask your web developer if they can integrate this tool.

## Security & Best Practices

### Keep Your Information Safe

1. **Never share API keys** (if you set one up)
2. **Never put API keys in code** you publish on GitHub
3. **Keep API keys in the `.env` file** (it's excluded from GitHub)
4. **If you accidentally share a key:**
   - Delete it immediately at https://platform.openai.com
   - Create a new one

---

### For Your Website

If you add this tool to your website:
- Only expose the "read-only" tools (list_services, generate_quote, calculate_payment)
- Don't expose tools that modify data
- Use HTTPS (encrypted) when collecting data

## Troubleshooting

### Problem: ChatGPT doesn't know about my tool

**Solution:**
1. Make sure the tool is running (`python server.py` in terminal)
2. Paste the setup instructions into ChatGPT again
3. Wait 30 seconds
4. Ask a test question

---

### Problem: Quote generated has wrong prices

**Solution:**
1. Open `server.py` in a text editor
2. Find your pricing
3. Update it
4. Save the file
5. Restart the server

---

### Problem: The chatbot isn't connecting to my website

**Solution:**
1. Ask your web developer to check the connection
2. Make sure the tool is running
3. Check that paths are correct in the configuration
