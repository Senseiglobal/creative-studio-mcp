Creative Studio MCP

A simple start page for people who want the tool installed, checked, and ready to connect without long technical instructions.

After install guide
First extract the ZIP file. Do not run this page from inside the ZIP preview or a temporary folder.
1

Install

Open the project folder and double-click install.bat on Windows.

Wait until you see Installation complete.
2

Confirm

Double-click CHECK_INSTALL.bat to confirm the tool is installed correctly.

If it says Success, the setup is fine.
3

Connect

Double-click CONNECT_CLAUDE.bat, restart Claude Desktop, then ask a test question.

No manual Claude settings needed on Windows.

Windows Users: Do This

This is the easiest path. No typing. No activation. Just double-click the files in order.

Before you begin: right-click the downloaded ZIP file and choose Extract All. Open the extracted folder before double-clicking anything.


1. Install: double-click install.bat.

2. Check: double-click CHECK_INSTALL.bat.

3. Connect: double-click CONNECT_CLAUDE.bat.

4. Restart Claude: fully close Claude Desktop, then open it again.

5. Test: ask Claude, What services do we offer?
