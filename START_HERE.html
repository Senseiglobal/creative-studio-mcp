Windows install Quick start Installer guide
1

Install

Open the project folder and double-click install.bat on Windows.

The installer sets up the project and checks that the tool can load.
2

Confirm

Wait for the message that says Installation complete.

If there is an error, read the message on screen. It will tell you what is missing.
3

Connect

Open QUICK_START.md and connect the tool to Claude Desktop or your AI assistant.

After connection, ask your AI to create a quote, payment breakdown, or checklist.

Choose Your Computer

Pick the option that matches your computer. Use the copy button if you need the command.

Best option: double-click install.bat.
