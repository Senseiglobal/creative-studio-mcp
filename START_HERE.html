Creative Studio MCP Setup

Extract the ZIP file first. Then open the creative-studio-mcp folder.

One File Setup

SETUP_WINDOWS.bat

Double-click this file. It installs the tool and connects Claude Desktop.

1. Install

Double-click SETUP_WINDOWS.bat.

2. Restart Claude

Fully quit Claude Desktop and open it again.

3. Test

Ask Claude to use the Creative Studio MCP tool to list services.

If Claude Cannot See It

Double-click CONNECT_CLAUDE.bat, then fully quit and reopen Claude Desktop.
