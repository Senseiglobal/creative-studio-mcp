# Safe Download Guide

This guide helps non-technical users avoid unsafe downloads.

## Safe Steps

1. Download from the official GitHub repo only.
2. Extract the ZIP file.
3. Open the folder.
4. Double-click `SECURITY_CHECK.bat`.
5. If the check looks normal, double-click `START_APP.bat`.

## Do Not Run It If

- It came from a random website.
- Someone sent you a private modified ZIP.
- The file names look strange.
- Antivirus blocks it.
- It asks for admin access.
- It asks you to disable Windows security.

## Private Files

Do not share:

- `.env`
- `brand_profile.json`
- `projects.json`
- `deleted_projects.json`
- `exports`

These files may contain your business or client information.
